# Create Ganesha Pooja Template

## Option 1: Via Admin Panel (Recommended)

1. **Login as Admin**: Go to http://localhost:3000/admin
2. **Navigate to Templates**: Look for "Templates" or "Sankalpam Templates" section
3. **Create New Template** with these details:
   - **Name**: "Ganesha Pooja Sankalpam (Telugu)"
   - **Description**: "Complete Ganesha Pooja Sankalpam with all rituals"
   - **Language**: "Telugu"
   - **Template Text**: (See below)

## Option 2: Via API (Using Swagger UI)

1. **Open API Docs**: http://localhost:8000/docs
2. **Login first**: Use `/api/auth/login` to get a token
3. **Authorize**: Click "Authorize" button and enter: `Bearer YOUR_TOKEN`
4. **Create Template**: Use `/api/admin/templates` POST endpoint

## Variables Used

- `{{user_name}}` - User's full name
- `{{gotram}}` - User's gotram
- `{{family_members}}` - Family members list
- `{{geographical_reference}}` - Geographic reference (e.g., "జంబూద్వీపే భారతవర్షే")
- `{{location_city}}` - Current city
- `{{location_state}}` - Current state
- `{{location_country}}` - Current country
- `{{nearby_river}}` - Nearby river name
- `{{samvathsarE}}` - Year name
- `{{ayanam}}` - Solstice direction
- `{{rithou}}` - Season
- `{{mAsE}}` - Month
- `{{pakshE}}` - Lunar phase
- `{{thithou}}` - Lunar day
- `{{vAsara}}` - Day of week
- `{{nakshatra}}` - Star constellation
- `{{yoga}}` - Auspicious combination
- `{{pooja_name}}` - Pooja name (e.g., "Ganesha Pooja")
